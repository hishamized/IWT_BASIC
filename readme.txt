Follow these steps
1) Clone or download this repository as a zip file.
2) Extract the zip file into C:\xampp\htdocs
3) Install XAMPP (with apche, mysql and PHP)
4) Create a database named "iwt" in mysql
5) Import the iwt.sql file located at "IWT_BASIC\database\iwr.sql"
   or
   create tables yourself using the queries given at "IWT_BASIC\database\database.txt"
6) Done!

NOTE: This repository included all the code for JavaScript form validation, Connecting PHP web app to mySQL server, inserting data into mySQL database and PHP Image Uploading.
